How to Grant Access
In your Firebase console, open the user's document under users / {uid} and set:

role: "admin" — full dashboard access
role: "ambassador" — activity tab only

No code changes needed. Access is checked on every login.
Manually Upgrade a User

Enter the user's UID (tap a user row to copy it) to manually set their plan to Premium.
